# Chemius (chemius)

Chemius is a cloud-based chemical compliance platform that automates Safety Data Sheet (SDS), Technical Data Sheet (TDS), and regulatory label creation in 38+ languages for organizations handling chemical products. The platform exposes an API suite covering SDS, TDS, ADR transport documentation, ERP integration, labels, web shop product data, and PDF generation, and offers AI-assisted authoring through the Chemius AI SDS assistant. Chemius is hosted in DIN ISO/IEC 27001-certified German data centers and supports CLP 1272/2008, REACH 1907/2006, detergents, aerosols, and US OSHA / GHS regulatory frameworks.

## APIs

### Chemius Safety Data Sheet API

Programmatic access to Chemius SDS creation, retrieval, and version control with multilingual output aligned with CLP, REACH, and GHS.

**Human URL:** [https://www.chemius.net/chemius-api/](https://www.chemius.net/chemius-api/)

### Chemius Technical Data Sheet API

API for generating and retrieving Technical Data Sheets for chemical products with version control and translation.

### Chemius ADR Transport API

API for ADR transport documentation for dangerous goods.

### Chemius ERP Integration API

API for syncing chemical product, SDS, TDS, and label data with enterprise ERP systems.

### Chemius Label API

API for generating regulatory-compliant chemical labels with QR codes, hazard pictograms, and multilingual content.

### Chemius Web Shop API

API exposing chemical product data, SDSs, and TDSs for embedding in e-commerce experiences.

### Chemius PDF API

API for rendering Chemius compliance documents as PDF artifacts.
